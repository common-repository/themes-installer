Il tuo host non ti permette di installare i temi? Ecco il plugin che fa per te!

== Description ==

Dedicato a chi usa un free host per wordpress che non permette di installare i temi dal proprio computer, soprattutto per chi usa l'AlterBlog ( [altervista](http://it.altervista.org "altervista") ).

Questo plugin ti permette di gestire facilmente i temi sul tuo blog.

Puoi installarli ed eliminarli in assoluta libert&agrave;

[Pagina di Supporto](http://javascripter.altervista.org/themes-installer/ "Pagina di Supporto")

== Installation ==

1) Scaricate il plugin
2) Caricatelo su wordpress ( Plugin -> Aggiungi Nuovo -> Carica Media )
3) Attivatelo
4) (Andate nella pagina Impostazioni -> Installa temi
